Methodology: Four Question Types × Four Death Conditions

Question Type 1
"Does X hold?"
Binary with escape routes. Branches: Yes / No / Meaningless / Illusory. Max 4 branches.
Question Type 2
"What IS X?"
Ontological identity. Run every philosophical category: material / mental / informational / relational / mathematical / brute / composite.
Question Type 3
"What grounds X?"
Explanatory origin. Branches: nothing (brute) / external (regress risk) / itself (self-grounding) / illusory / necessary.
Question Type 4
"What must X be?"
Constraint derivation. Not branching — deduction. Given survivors, derive necessary properties.
