# Semantic Co-Partner Protocol Pack

This doc pack is the Forge implementation starter for belief-to-evidence sessions.

## Files
- `front_questions.md`: user-visible question flow
- `hidden_ai_protocol.md`: non-visible AI routine
- `session_output.schema.json`: machine-readable output contract
- `system_prompt_template.md`: prompt for online AI sessions

## Intent
Support co-reasoning:
- user keeps decision authority
- AI must challenge weak reasoning
- system logs everything without manual note-taking
